# iOS Developer Portfolio

A clean, minimal, and professional portfolio website for an iOS App Developer with UI/UX design background.

## Features

- **Profile Section**: Concise, professional introduction customized for iOS development and UI/UX design
- **Skills Section**: Visual skill tags showcasing technical expertise
- **Work Samples Section**: GitHub links and project showcases
- **Education & Acknowledgements Section**: Academic credentials, certificates, transcripts, and recommendations
- **Resume Download**: Download button in navigation menu for PDF resume
- **Smooth Navigation**: Fixed navigation menu with jump links to all sections
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Clean & Minimal**: Modern, professional design with excellent typography

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory, ready to be deployed to any static hosting service.

## Deployment

You can deploy this portfolio to:
- **Netlify**: Connect your GitHub repo or drag and drop the `dist` folder
- **Vercel**: Import your project and deploy
- **GitHub Pages**: Use the `gh-pages` package or GitHub Actions
- **Google Sites**: Export and upload the built files
- **Any static hosting service**: Upload the `dist` folder contents

## Customization

1. **Add Your Resume**: Place your resume PDF file in the `public` folder and name it `resume.pdf`
2. Update the GitHub link in `src/components/WorkSamples.jsx`
3. Add your actual project details and links
4. Update education information in `src/components/Education.jsx`
5. Add your certificates, transcripts, and recommendations
6. Customize colors and styling in the CSS files
